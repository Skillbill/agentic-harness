# S01: INDEX + load on-demand (foundation)

**Goal:** Sostituire l'iniezione forzata di `.pi/codebase/*.md` interi nell'estensione (`index.ts:119-159`) con (a) un INDEX compatto `path: one-line summary` iniettato una sola volta a sessione e (b) un tool `load_codebase_doc({ name })` registrato via `pi.registerTool` che il LLM chiama on-demand per ricevere il contenuto integrale di un singolo doc. L'INDEX e il tool diventano l'unico canale autoritativo per leggere `.pi/codebase/`; le 4 skill esistenti continuano a funzionare invariate (loro modifica Ă¨ S02).
